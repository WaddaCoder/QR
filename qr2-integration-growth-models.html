Basic Integration Formulas — Power Rule, 1/x, and Exponential
Language First

Integration is the reverse of differentiation. Differentiation asks: what is the rate of change of this function? Integration asks: what function has this as its rate of change? The result of indefinite integration is always a family of functions differing by a constant C — the constant of integration.

∫f(x)dx reads as "the integral of f(x) with respect to x." The indefinite integral gives a function + C. The definite integral ∫ₐᵇ f(x)dx gives a specific number — the accumulated total from a to b.

Why C Matters in Growth Problems

Every growth model produces a constant C when you integrate. You eliminate C by applying the initial condition — the known value at t = 0 (the starting population P₀). The initial condition is not optional — it is what makes the general integral specific to your problem.

Mechanics & Notation
Power Rule: ∫xⁿ dx = xⁿ⁺¹/(n+1) + C    (n ≠ -1)
Constant Rule: ∫k dx = kx + C
Natural Log: ∫(1/x) dx = ln|x| + C    (the n = -1 exception)
Exponential: ∫eˣ dx = eˣ + C
Scaled Exponential: ∫eᵏˣ dx = eᵏˣ/k + C
Sum Rule: ∫[f(x)+g(x)] dx = ∫f(x)dx + ∫g(x)dx
Constant Multiple: ∫k·f(x) dx = k·∫f(x) dx
The Most Tested Integral — ∫(1/P) dP = ln|P| + C

The power rule fails at n = −1 because you get division by zero in the exponent. This specific case is handled by the natural log rule. It is the left-hand side of every exponential growth derivation. Know it without hesitation.

Worked Examples
Easy — Apply the Three Core Rules

Find: (a) ∫x³ dx   (b) ∫5 dx   (c) ∫(1/x) dx

(a) Power rule, n = 3: x³⁺¹/(3+1) + C = x⁴/4 + C

(b) Constant rule: 5x + C

(c) Natural log exception: ln|x| + C

Verification by differentiation: d/dx(x⁴/4) = x³ ✓   d/dx(ln|x|) = 1/x ✓. Always verify by differentiating your answer back.

Medium — Mixed Expression with Initial Condition

Find y given dy/dx = 3x² − 2x + 5, with y(0) = 4.

Step 1 — Integrate: y = ∫(3x² − 2x + 5) dx = 3(x³/3) − 2(x²/2) + 5x + C = x³ − x² + 5x + C

Step 2 — Apply initial condition y(0) = 4: 0 − 0 + 0 + C = 4, so C = 4.

Solution: y = x³ − x² + 5x + 4. The initial condition eliminates C and gives the unique function passing through the given point. Every population growth problem uses this same two-step protocol.

Hard — The Two Growth-Model Integrals

Find: (a) ∫(1/P) dP   (b) ∫eᵏᵗ dt with k = 0.03   (c) ∫(3e²ᵗ − 2/t) dt

(a) ∫(1/P) dP = ln|P| + C. This is the left-hand result in the exponential growth derivation. It comes from ∫P⁻¹ dP — do NOT use the power rule here.

(b) ∫e⁰·⁰³ᵗ dt = e⁰·⁰³ᵗ/0.03 + C = (1/0.03)e⁰·⁰³ᵗ + C. This is the right-hand result after separating dP/dt = 0.03P.

(c) ∫(3e²ᵗ − 2/t) dt = 3(e²ᵗ/2) − 2ln|t| + C = (3/2)e²ᵗ − 2ln|t| + C. Two rules in one integral — handle each term separately.

INT 02
Separation of Variables — The Technique Behind Every Growth Model
Language First

Separation of variables is a technique for solving differential equations where the right-hand side can be written as a product of a function of x alone and a function of y alone. The strategy: rearrange so all y terms (including dy) are on the left, all x terms (including dx) are on the right, then integrate both sides independently.

This is the technique Dr. Tariq used to derive P(t) = P₀eᵏᵗ from dP/dt = kP, and P(t) = K/(1+Ae⁻ʳᵗ) from the logistic equation.

The Key Move — Treating dy/dx as a Fraction

Formally dy/dx is a limit, not a fraction. But for this technique, we treat it as one and multiply both sides by dx to separate. When you see dP/dt = kP, write it as (1/P) dP = k dt, then integrate both sides. This separation step is what the exam expects you to show.

Mechanics & Notation
General form: dy/dx = f(x) · g(y)
Step 1 — Separate: (1/g(y)) dy = f(x) dx
Step 2 — Integrate both sides: ∫(1/g(y)) dy = ∫f(x) dx
Step 3 — Solve for y (exponentiate if ln appeared)
Step 4 — Apply initial condition to find C

The core growth pattern:
dP/dt = kP → (1/P) dP = k dt → ∫(1/P) dP = ∫k dt
→ ln|P| = kt + C → P = eᶜ · eᵏᵗ = P₀eᵏᵗ
Worked Examples
Easy — Four Steps from Start to Finish

Solve dy/dx = 3y, with y(0) = 2.

Step 1 — Separate: (1/y) dy = 3 dx

Step 2 — Integrate: ∫(1/y) dy = ∫3 dx → ln|y| = 3x + C

Step 3 — Exponentiate: |y| = e³ˣ⁺ᶜ = eᶜ · e³ˣ. Since y > 0: y = Ae³ˣ (where A = eᶜ)

Step 4 — Initial condition y(0) = 2: 2 = Ae⁰ = A → A = 2.

Solution: y = 2e³ˣ. Verify: dy/dx = 6e³ˣ = 3(2e³ˣ) = 3y ✓

Medium — The Exponential Growth Model Derived

Solve dP/dt = 0.05P with P(0) = 1000. This is the exact equation from the textbook.

Step 1: (1/P) dP = 0.05 dt

Step 2: ∫(1/P) dP = ∫0.05 dt → ln|P| = 0.05t + C

Step 3: P = Ae⁰·⁰⁵ᵗ

Step 4: P(0) = A = 1000 → P(t) = 1000e⁰·⁰⁵ᵗ

At t = 10: P(10) = 1000e⁰·⁵ = 1000(1.6487) ≈ 1,649. At 5% continuous growth, the population grows by 64.9% over 10 years.

Hard — Two Models That Look Similar but Require Different Approaches

Compare: (A) dP/dt = k (constant rate) vs (B) dP/dt = kP (proportional rate). Both with P(0) = P₀.

Model A — Integrate directly: dP/dt = k → P = ∫k dt = kt + C. Apply P(0) = P₀: C = P₀. Solution: P(t) = P₀ + kt (linear).

Model B — Separate: (1/P)dP = k dt → ln P = kt + C → P = Aeᵏᵗ. Apply P(0) = P₀: A = P₀. Solution: P(t) = P₀eᵏᵗ (exponential).

Long-run comparison with k = 0.05, P₀ = 100 at t = 50: Model A gives 100 + 0.05(50) = 102.5. Model B gives 100e²·⁵ = 1,218. The difference — 2.5 vs 1,218 — is enormous. Choosing the wrong model in a damages projection changes the award by orders of magnitude at large t. This is why the model choice is always legally contested.

INT 03
Exponential Growth Model — dP/dt = kP and P(t) = P₀eᵏᵗ
Language First

The exponential growth model assumes: the rate of growth is proportional to the current size. The more you have, the faster you grow. This is the Malthusian model. Its differential equation is dP/dt = kP.

Three model names appear in the textbook: Exponential (P₀eʳᵗ), Malthusian (same formula, named after Thomas Malthus), and Logistic (bounded by K). The exponential derivation from dP/dt = kP is what the exam tests.

Continuous vs Discrete Rate

P(t) = P₀eᵏᵗ uses a continuous rate k. P(t) = P₀(1+r)ᵗ uses a discrete annual rate r. They relate by: k = ln(1+r), or equivalently eᵏ = 1+r. For the same nominal rate, the continuous model always gives a slightly higher result because it compounds at every instant. On an exam: use whichever form the problem specifies. If neither is specified, use continuous.

Mechanics & Notation
Differential equation: dP/dt = kP   (k > 0 growth, k < 0 decay)
Solution: P(t) = P₀eᵏᵗ
Doubling time: T = ln(2)/k ≈ 0.693/k
Rule of 70: T ≈ 70/r%   (e.g. 5% → T ≈ 14 years)
Find k from two data points: k = ln(P₂/P₁) / (t₂ − t₁)
Find t when P reaches P*: t = ln(P*/P₀) / k
Continuous vs discrete: k = ln(1+r)   r = eᵏ − 1

INT 04
Logistic Growth Model — dP/dt = rP(1−P/K) and the S-Curve
Language First

The exponential model grows forever. Real populations don't — they face limited resources. The logistic model fixes this with a carrying capacity K: growth slows as P approaches K, and stops exactly at K.

The differential equation dP/dt = rP(1 − P/K) captures this: when P is small, (1 − P/K) ≈ 1 and growth is approximately exponential. As P → K, the brake term (1 − P/K) → 0 and growth stops.

The S-Curve

The logistic model produces an S-shaped curve: slow start, rapid exponential middle, deceleration approaching K. The fastest growth (inflection point) is at P = K/2. The Verhulst model in the textbook modifies the brake term: dP/dt = rP(1−P/K)² — same idea, steeper deceleration. For the exam, the standard logistic is the core.

Mechanics & Notation
Differential equation: dP/dt = rP(1 − P/K)
Parameters: r = intrinsic growth rate   K = carrying capacity
Solution: P(t) = K / (1 + Ae⁻ʳᵗ)
Constant A: A = (K − P₀) / P₀
Equilibria: P = 0 (unstable) and P = K (stable)
Inflection point (fastest growth): P = K/2
Max growth rate: dP/dt|_{P=K/2} = rK/4
Time of inflection: t* = ln(A)/r
Verhulst (textbook variant): dP/dt = rP(1−P/K)²
Worked Examples

Find A: A = (1000 − 500)/500 = 500/500 = 1

Medium — Inflection Point and Maximum Growth Rate

(a) A = (1000−100)/100 = 9. P(t) = 1000/(1+9e⁻⁰·²ᵗ)

(b) Fastest growth at P = K/2 = 500. When: 1000/(1+9e⁻⁰·²ᵗ) = 500 → 1+9e⁻⁰·²ᵗ = 2 → 9e⁻⁰·²ᵗ = 1 → e⁻⁰·²ᵗ = 1/9 → −0.2t = ln(1/9) = −2.197 → t* = 10.99 years.

Before year 11: growth accelerates (S-curve rising steeply). After year 11: growth decelerates (S-curve flattening toward K=1000).

Hard — Derive the Logistic Solution via Partial Fractions

Starting from dP/dt = rP(1−P/K), derive P(t) = K/(1+Ae⁻ʳᵗ).

Step 1 — Separate: dP / [P(1−P/K)] = r dt. Rewrite: K dP / [P(K−P)] = r dt.

Step 2 — Partial fractions on K/[P(K−P)]: Write K/[P(K−P)] = A/P + B/(K−P). Multiply through: K = A(K−P) + BP. Set P=0: K = AK → A=1. Set P=K: K = BK → B=1. So: 1/P + 1/(K−P).

Step 3 — Integrate: ∫[1/P + 1/(K−P)] dP = ∫r dt → ln|P| − ln|K−P| = rt + C → ln[P/(K−P)] = rt + C.

Step 4 — Exponentiate: P/(K−P) = Be^(rt) (where B = eᶜ).

Step 5 — Solve for P: P = BKe^(rt) / (1+Be^(rt)) = K / (1 + (1/B)e⁻ʳᵗ).

Step 6 — Initial condition P(0)=P₀: P₀ = K/(1+1/B) → 1/B = (K−P₀)/P₀ = A. P(t) = K/(1+Ae⁻ʳᵗ). ✓

The exam will likely give you the formula and ask you to apply it, or ask you to show the separation and partial fractions steps. The full six-step derivation above is the complete picture.

INT 05
Exponential Decay — dP/dt = −kP, Half-Life, and Legal Applications
Language First

Exponential decay is the mirror image of exponential growth: instead of growing proportionally to current size, a quantity shrinks proportionally to current size. The differential equation is dP/dt = −kP where k > 0 — the negative sign is what makes it decay rather than grow.

Everything that grows exponentially can also decay exponentially, and the mathematics is identical — only the sign of the rate constant changes. Legal applications include radioactive decay (forensic dating), drug concentration in blood (impairment cases), depreciation of asset value (damages), and decay of environmental contamination (liability timelines).

Half-Life — The Key Concept

The half-life T½ is the time for a quantity to reduce to exactly half its current value. It is constant — a quantity always takes the same amount of time to halve, regardless of how much remains. This constancy is what makes exponential decay distinctive and identifiable from data. Formula: T½ = ln(2)/k ≈ 0.693/k. It is the exact decay analogue of doubling time in growth.

Mechanics & Notation
Differential equation: dP/dt = −kP   (k > 0 always)
Solution (separation of variables): P(t) = P₀e^(−kt)
Derivation: (1/P)dP = −k dt → ln P = −kt + C → P = P₀e^(−kt)
Half-life: T½ = ln(2)/k ≈ 0.693/k
Find k from half-life: k = ln(2)/T½
Find t when P reaches P*: t = −ln(P*/P₀)/k = ln(P₀/P*)/k
After n half-lives: P = P₀ × (1/2)ⁿ
Fraction remaining: P(t)/P₀ = e^(−kt)
Worked Examples
Easy — Drug Concentration Decay

A drug has a half-life of 4 hours in the bloodstream. A patient takes a dose giving an initial concentration of 200 mg/L. (a) Find k. (b) Write C(t). (c) Find concentration after 12 hours.

(a) k = ln(2)/T½ = 0.693/4 = 0.1733/hour

(b) C(t) = 200e^(−0.1733t)

(c) After 12 hours = 3 half-lives: C = 200 × (1/2)³ = 200 × 0.125 = 25 mg/L. Or: C(12) = 200e^(−0.1733×12) = 200e^(−2.079) = 200(0.125) = 25 mg/L ✓. Legal relevance: if the legal impairment threshold is 50 mg/L, the patient clears the threshold at T where 200e^(−0.1733T) = 50 → e^(−0.1733T) = 0.25 → −0.1733T = ln(0.25) = −1.386 → T = 8 hours = exactly 2 half-lives.

Medium — Forensic Dating: Find Time from Remaining Fraction

A piece of organic evidence contains 72% of its original Carbon-14. C-14 half-life is 5,730 years. Find k and estimate the age of the evidence.

Find k: k = ln(2)/5,730 = 0.693/5,730 = 0.0001210/year

Find t: P(t)/P₀ = 0.72 → e^(−0.0001210t) = 0.72 → −0.0001210t = ln(0.72) = −0.3285 → t = 0.3285/0.0001210 = 2,715 years.

The evidence is approximately 2,715 years old. A defence claiming the evidence is less than 1,000 years old would require 87.9% of C-14 remaining (e^(−0.1210×0.1) = 0.879) — inconsistent with the 72% measured. The decay model directly refutes this claim.

Hard — Asset Depreciation and Damages Calculation

A piece of industrial equipment worth Rs. 5,000,000 depreciates exponentially at 15% per year. It is damaged in an accident after 3 years. (a) Derive the value function V(t). (b) Find the value at the time of damage. (c) The plaintiff claims damages of the original purchase price (Rs. 5M). The defendant argues for replacement cost at the time of damage. Compute both and explain which is the legally appropriate measure.

(a) Derivation: dV/dt = −0.15V → V(t) = 5,000,000e^(−0.15t)

(b) V(3): 5,000,000e^(−0.45) = 5,000,000(0.6376) = Rs. 3,188,000.

(c) Damages comparison: Plaintiff claims Rs. 5,000,000 (original cost). Defendant argues Rs. 3,188,000 (market value at time of damage). The legally appropriate measure in most tort frameworks is the diminution in value — restoring the plaintiff to the position they were in before the damage, not improving it. The asset was worth Rs. 3,188,000 at the time of damage; that is the true loss. Awarding Rs. 5,000,000 would overcompensate — the plaintiff would be better off than before the damage (since the asset had already lost Rs. 1,812,000 of value through normal depreciation). The defendant's figure is the legally correct measure. Note: if the plaintiff can show the equipment was irreplaceable and had unique productive value beyond its market price, additional consequential damages may be argued separately.

Practice Questions
T2 · AA radioactive substance has k = 0.0347/year. (a) Find the half-life. (b) Write P(t) with P₀ = 100g. (c) Find P after 40 years. (d) Find when P = 10g.

(a) T½ = ln(2)/0.0347 = 0.693/0.0347 = 20 years.

(b) P(t) = 100e^(−0.0347t)

(c) 40 years = 2 half-lives: P(40) = 100 × (1/2)² = 25g. Or: 100e^(−1.388) = 100(0.249) = 25g ✓

(d) 100e^(−0.0347t) = 10 → e^(−0.0347t) = 0.10 → −0.0347t = ln(0.1) = −2.303 → t = 2.303/0.0347 = 66.4 years ≈ 3.32 half-lives.

T½=ln(2)/k · P(t)=P₀e^(−kt) · n half-lives shortcut · logarithm for threshold time
T2 · BAlcohol in blood decays exponentially. A driver has a blood alcohol level (BAC) of 0.18% at the time of an accident. The legal limit is 0.08%. The body eliminates alcohol at k = 0.15/hour. (a) Write BAC(t). (b) How long before the driver reaches the legal limit? (c) The driver claims they had stopped drinking 2 hours before the accident and the BAC at the time of driving was legal. What was the BAC 2 hours before the accident?

(a) BAC(t) = 0.18e^(−0.15t)

(b) 0.18e^(−0.15t) = 0.08 → e^(−0.15t) = 0.444 → −0.15t = ln(0.444) = −0.811 → t = 5.4 hours to reach legal limit from 0.18.

(c) BAC 2 hours earlier: at t = −2 (2 hours before the accident), BAC = 0.18e^(−0.15×(−2)) = 0.18e^(+0.30) = 0.18 × 1.350 = 0.243%. The driver's claim is false — 2 hours before the accident, their BAC was 0.243%, far above the legal limit. The driver would have had to stop drinking many hours before the accident for their BAC to have been legal at the time of driving. At 0.18% post-accident, working backward 2 hours gives 0.243% — three times the legal limit at the claimed "time of driving."

BAC(t)=0.18e^(−0.15t) · time to legal limit 5.4 hours · back-calculation: e^(+0.30) multiplier · driver's claim refuted by back-calculation
T2 · CDistinguish exponential decay from linear decay. A contamination level starts at 500 ppm. Model A: linear decay at 20 ppm/year. Model B: exponential decay at k = 0.05/year. (a) Write both models. (b) Compute contamination at t = 5, 10, 20 years. (c) When does each model predict the contamination reaches the legal limit of 100 ppm?

(a) Linear: C_L(t) = 500 − 20t. Exponential: C_E(t) = 500e^(−0.05t).

(b) t=5: C_L=400, C_E=500e^(−0.25)=500(0.779)=389. t=10: C_L=300, C_E=500e^(−0.5)=500(0.607)=303. t=20: C_L=100, C_E=500e^(−1)=500(0.368)=184. At t=20 the models diverge significantly.

(c) Linear: 500−20t=100 → 20t=400 → t=20 years. Exponential: 500e^(−0.05t)=100 → e^(−0.05t)=0.2 → −0.05t=ln(0.2)=−1.609 → t=32.2 years. The exponential model predicts compliance takes 12.2 years longer. A company would prefer the linear model (shorter timeline); a regulator would use the exponential model for a conservative estimate. Which model fits better must be determined from actual measured decay data.

Linear C=500−20t vs exponential C=500e^(−0.05t) · values at t=5,10,20 · legal limit: 20 yrs linear vs 32.2 yrs exponential · model choice determines compliance timeline
T2 · DWhat is half-life? Why is constant half-life a defining feature of exponential decay? Demonstrate that the half-life is constant for P(t) = P₀e^(−kt) by showing that the time from P₀ to P₀/2 equals the time from P₀/2 to P₀/4.

Half-life T½: the time required for a quantity to reduce to exactly half its current value. Constant half-life means: no matter how much remains, it always takes the same amount of time to halve again. This distinguishes exponential decay from linear decay (where the time to halve increases as less remains) and from other models.

Proof of constant T½: Time from P₀ to P₀/2: P₀e^(−kt₁) = P₀/2 → e^(−kt₁) = 1/2 → kt₁ = ln(2) → t₁ = ln(2)/k. Time from P₀/2 to P₀/4: (P₀/2)e^(−kt₂) = P₀/4 → e^(−kt₂) = 1/2 → kt₂ = ln(2) → t₂ = ln(2)/k. Since t₁ = t₂ = ln(2)/k, the half-life is constant regardless of starting amount. ✓ This is why the half-life formula T½ = ln(2)/k can be used at any stage of decay, not just from the initial value.

Half-life definition · constant T½ is the defining property · proof: time P₀→P₀/2 equals time P₀/2→P₀/4 · both equal ln(2)/k
T3 · AA pesticide is found in groundwater at 800 ppb (parts per billion). The legal maximum is 50 ppb. The pesticide decays exponentially with T½ = 6 months. (a) Write C(t) and find k. (b) When will the water be legal to use? (c) A farmer demands compensation for every month until the water is legal. Compute total compensation if loss = Rs. 10,000 × C(t)/C(0) per month (proportional to contamination). (d) The company argues they should only pay until contamination reaches 100 ppb (half the reduction), after which "natural decay handles it." Evaluate this argument legally and mathematically.

(a) k = ln(2)/6 = 0.1155/month. C(t) = 800e^(−0.1155t) ppb.

(b) 800e^(−0.1155t) = 50 → e^(−0.1155t) = 0.0625 = 1/16 → −0.1155t = ln(1/16) = −4ln(2) = −2.773 → t = 2.773/0.1155 = 24 months = 4 half-lives. Check: 800 → 400 → 200 → 100 → 50 after 4 half-lives ✓

(c) Total compensation = ∫₀²⁴ 10,000 × (C(t)/800) dt = ∫₀²⁴ 10,000 × e^(−0.1155t) dt = 10,000 [e^(−0.1155t)/(−0.1155)]₀²⁴ = (10,000/0.1155)(1 − e^(−2.773)) = 86,580 × (1 − 0.0625) = 86,580 × 0.9375 = Rs. 81,169.

(d) Mathematical analysis: at t=12 months (2 half-lives), C=200 ppb. The company wants to stop paying at C=100 ppb — when does that occur? At t=18 months (3 half-lives): 800/8=100 ppb. So the company wants to pay for 18 months, not 24. Compensation under their proposal: ∫₀¹⁸ 10,000 × e^(−0.1155t) dt = (10,000/0.1155)(1−e^(−2.079)) = 86,580(1−0.125) = 86,580×0.875 = Rs. 75,758.

Legal evaluation: the company's argument is legally weak. The harm to the farmer is not the contamination level per se but the inability to use the water — which persists until the legal limit (50 ppb) is reached at month 24, not month 18. "Natural decay handles it" from month 18 is true only in the mathematical sense — but the farmer is still unable to use the water during that period. The legal liability runs until the harm (unusable water) ends, not until some arbitrary intermediate level. The company is liable for the full 24-month period. The difference: Rs. 81,169 − Rs. 75,758 = Rs. 5,411 disputed.

k=0.1155/month · legal at t=24 months = 4 half-lives · integral for total compensation Rs.81,169 · company's 18-month proposal = Rs.75,758 · Rs.5,411 difference · harm runs until legal limit not intermediate milestone
T3 · BTwo experts disagree on the decay constant for a financial asset. Expert P (plaintiff): k = 0.20/year. Expert D (defendant): k = 0.10/year. Asset value at time of damage (t=0): Rs. 1,000,000. Damages are based on lost value over 5 years (the asset would have been replaced after 5 years anyway). (a) Compute V(5) under each model. (b) Compute the present value of the total loss over 5 years under each model: Loss(t) = V(0) − V(t) per year, total = ∫₀⁵ (V(0)−V(t)) dt. (c) Show which expert benefits the plaintiff and by how much. (d) How should a court determine the correct k?

(a) Expert P: V_P(5)=1,000,000e^(−1.0)=1,000,000(0.368)=Rs.368,000. Expert D: V_D(5)=1,000,000e^(−0.5)=1,000,000(0.607)=Rs.607,000.

(b) Total loss = ∫₀⁵ (1,000,000 − 1,000,000e^(−kt)) dt = 1,000,000∫₀⁵ (1−e^(−kt)) dt = 1,000,000 [t + e^(−kt)/k]₀⁵ = 1,000,000[(5+e^(−5k)/k) − (0+1/k)] = 1,000,000[5 − (1−e^(−5k))/k].

Expert P (k=0.20): = 1,000,000[5−(1−e^(−1))/0.20] = 1,000,000[5−(0.632)/0.20] = 1,000,000[5−3.16] = Rs. 1,840,000.

Expert D (k=0.10): = 1,000,000[5−(1−e^(−0.5))/0.10] = 1,000,000[5−(0.393)/0.10] = 1,000,000[5−3.93] = Rs. 1,070,000.

(c) Expert P benefits the plaintiff by Rs.1,840,000 − Rs.1,070,000 = Rs. 770,000 more. Expert P (plaintiff's witness) uses a higher k (faster depreciation), meaning the asset was worth less by year 5, so its loss is calculated as more relative to a reference value. This seems counterintuitive — faster depreciation should mean the asset was already declining — but the integrated loss over 5 years is higher under faster depreciation because the asset loses value more rapidly throughout the period.

(d) Court determination of k: require empirical evidence of the asset's market value at multiple points in time before and after the damage (comparable asset transactions, manufacturer's depreciation tables, insurance company valuation schedules, auction results for similar assets). k should be estimated from actual market data, not asserted by either expert without empirical basis. If asset value data is available at t=0 and t=1, for example: k = −ln(V(1)/V(0)). The court should appoint a single independent financial expert if the two experts' figures cannot be reconciled through cross-examination.

V_P(5)=Rs.368K vs V_D(5)=Rs.607K · integrated loss Rs.1.84M vs Rs.1.07M · Rs.770K difference · higher k benefits plaintiff via integration · k must be empirically determined from market data
T3 · CA company discharged industrial waste into a river in 2010. The contaminant decays with T½ = 3 years. In 2016, the regulator finds contamination at 80 ppb. (a) Work backwards to estimate the 2010 discharge level. (b) In 2010, the legal discharge limit was 200 ppb. Did the company violate it? (c) The company argues: "We cannot be held liable — you have no measurement from 2010, only an estimate." Evaluate this argument legally and statistically. (d) What is the uncertainty in the back-calculated 2010 figure if the 2016 measurement has ±10 ppb error?

(a) t = 6 years from 2010 to 2016. k = ln(2)/3 = 0.2310/year. C(6) = C₀e^(−0.2310×6) = C₀e^(−1.386) = C₀ × 0.25 = 80. So C₀ = 80/0.25 = 320 ppb. Equivalently: 6 years = 2 half-lives, so contamination halved twice: 80 → 160 → 320.

(b) The back-calculated 2010 discharge level is 320 ppb. The legal limit was 200 ppb. 320 > 200: the company violated the limit by 60% above the legal maximum.

(c) The company's argument has procedural force but fails on the merits. Courts regularly accept back-calculated estimates as evidence where the underlying model is scientifically established and the inputs are measured. The exponential decay model for this contaminant type, and its half-life, are established physical chemistry — not opinion. The argument "only an estimate" could be raised against any forensic analysis (carbon dating, toxicology, blood alcohol back-calculation) and courts routinely accept such evidence when the method is reliable and the inputs are measured with stated uncertainty. The company would need to challenge the half-life figure or the accuracy of the 2016 measurement — not the concept of back-calculation itself.

(d) Measurement error propagation: C₀ = C(6)/e^(−1.386) = C(6)/0.25 = 4 × C(6). If C(6) = 80 ± 10, then C₀ = 4 × (80 ± 10) = 320 ± 40 ppb. Range: 280 to 360 ppb. Even the lower bound (280 ppb) exceeds the legal limit (200 ppb). The uncertainty does not rescue the company — even the most conservative back-calculation places the 2010 discharge 40% above the legal limit. The company's violation is robust to measurement error.

Back-calculation: 6 years = 2 half-lives → C₀=320 ppb · 320>200 legal limit violated · back-calculation admissible as established science method · error propagation: ±40 ppb · even lower bound 280>200 violation is robust

Decay Model
dP/dt=−kP (k>0) → P(t)=P₀e^(−kt) · T½=ln(2)/k≈0.693/k · After n half-lives: P=P₀×(1/2)ⁿ
Key Operations
Find k: k=ln(2)/T½ · Find t for P*: t=ln(P₀/P*)/k · Back-calculate P₀: P₀=P(t)×e^(kt) · Error propagates by same factor e^(kt)
Growth vs Decay
Growth: dP/dt=+kP → P=P₀e^(+kt) · Decay: dP/dt=−kP → P=P₀e^(−kt) · Same maths, sign of exponent differs · Doubling time T=ln(2)/k; Half-life T½=ln(2)/k (same formula)
